# Playwright Capture (개발/분석 전용)

## 목적
이 디렉터리는 **개발 및 분석 목적의 요청/응답 캡처**를 위한 보조 도구를 제공합니다.

- 로그인/조회/전송 과정의 네트워크 호출 패턴 관찰
- 어댑터 설계 전 계약(Contract) 가설 검증
- 장애 재현 시 요청 시퀀스 비교

## 운영 로직 미사용 원칙
다음 원칙을 반드시 지켜야 합니다.

1. 본 디렉터리의 스크립트는 **프로덕션 운영 로직에서 호출하지 않습니다**.
2. 캡처 스크립트는 **분석용 임시 실행**만 허용합니다.
3. 운영 배포 파이프라인(CI/CD)의 필수 단계로 포함하지 않습니다.
4. 캡처 결과를 운영 API 호출 코드로 직접 복사하지 않고, 반드시 adapter/contract 계층을 통해 재정의합니다.

## 데이터 취급 주의
- 실제 사용자 계정/토큰/쿠키를 문서나 저장소에 커밋하지 않습니다.
- 민감 헤더(Authorization, Cookie 등)는 마스킹합니다.
- 테스트 시에는 placeholder endpoint와 비식별 테스트 데이터만 사용합니다.
